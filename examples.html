Featured

Watch + defer + std/math

runtime

Small but very “MKS”: module import, watched state, deferred cleanup, and a visible flow of events. 

using "std/math";

var x =: 10;
watch x;

on change x ->
  Writeln("x changed to", x);
<-

defer ->
  Writeln("leaving scope");
<-

x =: square(x);
Writeln("result:", x);

Featured

Entity + extend array

object model

Lightweight entities plus built-in extension is one of the coolest parts of the language surface. 

entity User(name) ->
  init ->
    self.name =: name;
  <-

  method hi() ->
    Writeln("Hi", self.name);
  <-
<-

extend array ->
  method sum() ->
    var s =: 0;
    var i =: 0;

    while (i < self.len()) ->
      s =: s + self[i];
      i =: i + 1;
    <-

    return s;
  <-
<-

var u =: User("Ann");
u.hi();
Writeln([1,2,3].sum());
