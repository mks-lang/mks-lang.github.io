Featured

Entity + extend array

object model

Lightweight entities plus built-in extension is one of the coolest parts of the language surface. 

entity User(name) ->
  init ->
    self.name =: name;
  <-

  method hi() ->
    Writeln("Hi", self.name);
  <-
<-

extend array ->
  method sum() ->
    var s =: 0;
    var i =: 0;

    while (i < self.len()) ->
      s =: s + self[i];
      i =: i + 1;
    <-

    return s;
  <-
<-

var u =: User("Ann");
u.hi();
Writeln([1,2,3].sum());
