Minimal syntax

Simple, readable structure with clear control flow.

Custom runtime

Built from scratch in C with its own execution model.

Garbage collected

Mark-sweep GC integrated into the runtime.

Example

fnc fib(n) ->
    if (n < 2) ->
        return n;
    <-
    return fib(n - 1) + fib(n - 2);
<-

Writeln(fib(10));
